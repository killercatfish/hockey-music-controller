# Quick Start: Custom Start Times

## What Is It?

Set tracks to start at a specific time (e.g., skip the first 15 seconds). Perfect for:
- Tracks with long intros
- Songs that start too slow
- Cloud tracks that won't seek properly

## 3-Step Setup

### Step 1: Right-Click a Track
```
In your playlist, right-click (or Ctrl+Click on Mac) any track
```

### Step 2: Choose "Set Start Time"

## Pro Tips

- **Cloud tracks**: Set start times for tracks from Tune My Music that won't seek
- **Batch setup**: Set start times for all your stoppage tracks at once
- **Perfect timing**: Use seconds for precise control (e.g., `23` for 23 seconds)
- **Saved forever**: Start times persist between sessions

## Troubleshooting

**Track still starts at beginning?**
- Check that the indicator `‚è±Ô∏è [0:XX]` is shown
- Try setting the time again
- Make sure the track name matches exactly

**Start time not saved?**
- Check that `~/hockey_music_config.json` exists and is writable
- Restart the app to reload config

**Wrong time?**
- Right-click ‚Üí "Edit Start Time"
- Or remove and set again
